Maintenance Instructions for MyHigh5 Application

To maintain the software life cycle of the MyHigh5 application, follow these steps:

  • Regularly update the code from the repository.
  • Update backend and frontend dependencies.
  • Run database migrations after updating the code.
  • Restart the backend, frontend, and Celery workers after updates.
  • Monitor application logs and performance.
  • Backup the database and media files regularly.
  • Check for and apply security updates.
  • Review and update environment variables as needed.
  • Test the application in a staging environment before deploying to production.
